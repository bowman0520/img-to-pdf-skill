# img-to-pdf

将多张图片一键合并为 PDF 的 Claude Skill。

## 核心功能

- 自动读取 `input/` 目录下的所有图片（支持 JPG、PNG 格式）
- 按自然顺序排序（1.png → 10.png → 100.png）
- 合并为单份 PDF 文件
- 保持原图比例，不压缩画质

## 目录规范

```
img-to-pdf/
├── input/          # 原材料区：放入待转换的图片
├── output/         # 成品区：生成的 PDF 输出到此
├── img_to_pdf.py   # 核心脚本
└── README.md       # 本文档
```

> **注意**：`input/` 和 `output/` 为固定目录，脚本自动定位，无需手动指定路径。

## 处理逻辑

1. **自然排序**：文件名中的数字按数值排序（如 1, 2, 10 而非 1, 10, 2）
2. **格式支持**：自动识别 `.jpg`、`.jpeg`、`.png` 及其大写形式
3. **依赖检测**：首次运行时自动安装 `img2pdf` 库
4. **输出固定**：始终生成 `output/combined_result.pdf`

## 使用范例

对 Claude 说：

- "帮我把 input 里的图片转了"
- "运行 img-to-pdf"

或直接执行：

```bash
python img_to_pdf.py
```

## 维护者建议

- **定期清理**：转换完成后建议清空 `input/` 目录，避免下次误处理旧文件
- **文件命名**：建议在放入 `input/` 前按顺序重命名图片（如 01.jpg, 02.jpg...），确保合并顺序正确
- **备份原图**：脚本不会修改或删除原图，但建议保留原始图片备份
